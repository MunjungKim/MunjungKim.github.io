Clustering

K means clustering

  1. Initialize the center of k clusters
  2. Each data belongs to the cluster with the nearest mean
  3. Update the average value of the data beloning to each cluster to the center of the new cluster
  4. Interate 2) and 3)

Weaknesses of K means clustering

  1. The performance depends on the number of clusters.
  2. The performance also depends on the inital points.

Algorithm of SOM

SOM needs 2-D neurons. Each neurons has position and weight. Denote each neuron as \(n_i = (w_i,k_i)\).

  1. Each node’s weights \(w_i\) are initialized.
  2. A vector \(x_i\) is chosen at random from the set of training data.
  3. Every node is examined to calculate which one’s weights are most close the input vector. The closet node \(n_m =\argmin_{n_i} d(x_j, W(n_j))\)is called the winning node.
  4. Pick a set of neurons for which we will adapt the weight vector given a neighborhood funcntion. It is important that the weight that should be updated are given by \(d(k_i,k_m\) where \(k_m\) is the position of the winning node.
  5. The winner nonde and its topological neighborhood neurons are updated by flowing the update rule

Update rule

$$ n_{i}(t+1) = n_{i} (t) + \sum_x \eta(t) \cdot h(||n_m(x) - n_i||,t) \cdot [x - n_i(t)] $$

  • \(h()\) is the neighborhood function. A variety of neighborhood functions can be usued.

  • \(\eta\) is the learning rate. It is the factor that how much the node will move at time \(t\). \(\eta \in (0,1]\) and \(\eta(t+1)<\eta(t)\). Typically the exponential function is used. $$ \eta(t) = \eta_0 k^{-t/t_{max}(\log \eta_f/\eta_0)/(1/logK)} $$
