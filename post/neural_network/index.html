This article is written based on the lecture MATH532 by Prof. Hwang at POSTECH.

Introduction

Motivation of Neural Network

The motivation of neural network methodology at first hand, as we can easily find in its name, is the structure of brain and the way in which the brain performs. Brain consists of about \(10^11\) neurons and about \(10^15\) synapses. In early models, the neurons transmit its signal if the potential exceed a certain threshold.

Units(Perceptron)

  • Units are each node of neural network.
  • Units are connected by links.
  • Each link has a numeric weight.
  • Each unit can be a machine learning model.
  • In the unit, the input variables are multiplied with weights while passing through the dendrite. Inside the cell body, output variable is calculated.

Activation Function

  • Whether pass the output variable from one unit to the next unit is determined by the activation function. It represent the rate of potential firing in the cell.

  • Ex) Sigmoid: \( (1+e^{-x})^{-1} \) , ReLU: \(max(0,x)\) …

Neural Network

  • Neural networks are modeled as collections of units.
    • Units are connected in an acyclic graph.

Fully-connected layer

  • Fully-connected layer is the layer of the neural network such that units between two adjacent layers are fully pairwise connected and units within a single layer share no connections.

  • Fully-connected layer is the most common layer type.

  • Limitations

    • It requires too many parameters. For example, if we have \(32\) by \(32\) fixels with \(3\) channels(RGB) data, the size of the input layer becomes \( 32 \times 32 \times 3 \). Then, the neural network with only one hidden layer has \( (32 \times 32 \times 3)^2 \), which is about \(~ 10^6\) scale. This can cause overfitting and huge amount of comutational cost.
    • Putting all data into the input layer can makes the problem of curse of dimensionality. Also, in terms of image data, this does not consider the position of the object we are interested in.

Recurrent Neural Network (RNN)

Motivation

  • Previous neural networks are all have fixed size of input nodes. However, there are some data whose size or length may not be fixed. (e.g. Text data, Time series data, …)

  • Previous inputs were not considered when the model generates the current output but previous one may affect the current output in real data.

RNN

  • The RNN is called recurrent becuase it applies the same task to every element of a sequence, an the output is affected by the previous computation.

  • Hidden state at time \(t\) is \(h_t = \sigma (W(x_t,h_{t-1})+b)\). This is also called the memory of the network, since it contains the history of the networks.

Back Propogation Through Time, BPTT

  • Back propogation means that parameters are updated based on its outcome in order to minimize the loss function.

  • We can use gradient descent method.

  • Limitation

    • If there are too many layers, gradient descent method requires a lot of matrix multiplications. Gradient contributions from “far away” steps become zero or exploding.
