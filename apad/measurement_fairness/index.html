Results

Identifying structural nodes in multiplex networks

Goal: to find the minimal set of nodes such that, if removed from the multiplex, no mutual cluster with a size greater than \(N^{1/2}\) is found in the network.

  • Finding approximate solution is not trivial. For instance, in the greedy search which many researchers have been used, node degrees is used to calculate the score for each node. Howeer, in the multiplex network, it is not easy to assign one score into one node.

  • Using various algorithms, the authors find optimal percolation solution with two layer multiplex network with \(N=10,000 \). Each network made through Erodoss-Renyi model with average k=5.

  • As a result, simulated annealing optimization strategy was the best. SA has variou advantages but it also has drawbacks such as computation cost

The size of the set of SNs

  • Relative error for optimal percolation behaves more or less in the same way as that of ordinary percolation. (inverse proportion)

  • Generally, multiplex networks have considerably smaller sets of SNs compared to the SN sets of their single-layer based network representations.

Edge overlap and degree correlations

  • Checked the dependence of the size of the SNs on overlap and layer to layer degree correlation.
  • The size of SNs deviates immediately after \(\alpha\) becomes bigger than 0. As soon as there is a finite fraction of edges that are not shared by the two layers, the system behaves exactly as a multiplex
  • The error committed when relying on single-layer representations of the multiplex does not regard only the size of the SN sets, but also the identity of the SNs.

Self Consistency

  • \(S = \sum_i p_{i}^2 /\sum_i p_{i} \) . Multiplex network with small overlap and correlation has small S

  • \(p_{i}\) is the probability that node \(i\) is found within the set of SNs of multiplex network. \(w_{i}\) is the probability that node \(i\) is found within the set of SNs of a specific single layer representation of that multiplex network.

  • Precision \(P = \sum_i p_{i} w_{i} /\sum_i w_{i} \). (i.e. the ratio of the expected number of correctly detected SNs by single layer to the to the expected total number of detected SNs. \(P\) deteriorates as the edge overlap and inter layer degree correlation decrease.

  • Recall \(R = \sum_i p_{i} w_{i} /\sum_i p_{i} \).

  • \(F_{1} = 2/(1/P +1/R) \)

Multiplex network

Multiplex network is a subset of multilayer network. Multiplex network has two properties.

  1. all layers share the same nodes.
  2. all nodes are connected themselves in the different layers. e.g.) diseases spreading when two pathogens each of which enhances or impairs the spreading of the other.

Percolation theory

Percolation indicates the phenomenon that the network is fragmented into small clusters when the critical fraction of nodes or links are randomly removed (node/edge isolation).

  • The size of connected components within a graph can be used to examine the robustests of the system to node damage or failure
  • Examine the capacity of the graph to maintain its connextedness under different kinds of topological attacks (graph topological perturbation).
  • At percolation threshold, the network is fragmented.(phase transition)
